FinSight — Smart Financial Dashboard

Plataforma financeira moderna desenvolvida com foco em experiência do usuário, acessibilidade, performance e visual premium inspirado em fintechs e bancos digitais modernos.

O FinSight foi criado para simular a experiência de plataformas utilizadas por empresas como XP Investimentos, Nubank, Itaú e Mercado Livre, unindo arquitetura Front-End moderna, dashboards inteligentes e UX Engineering.

Clone o repositório

Acesse a pasta
cd finsight-financial-dashboard
Instale as dependências
npm install
Rode o projeto
npm run dev
